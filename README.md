## tech stack

- **GSC** — the scripting language used to mod CoD games
- **IW8 Engine** — the engine behind Modern Warfare (2019)
- **iw8mod** — mod loader that lets you inject GSC into the game
- **gsc-tool** — used to compile and decompile GSC files
